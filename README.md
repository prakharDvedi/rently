## Rently

Rently is a platform for renting properties. It is a full-stack application that allows users within the same household to effeciently divide tasks, take turns and split bills

## Tech Stack

Next.js, TypeScript, Tailwind CSS, Prisma, PostgreSQL, AuthJS, Stripe

## Features

- User Authentication
- Property Management
- Payment Processing
- Booking System
- Search and Filtering
- Responsive Design

## Getting Started

- Clone the repository
- Install dependencies
- Set up environment variables
- Run the development server
- Open http://localhost:3000 in your browser
